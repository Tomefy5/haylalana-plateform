## [0.1.0] - 2026-01-03

### Added
- **Authentication System**
  - Login page with split-screen design
  - Registration flow with role selection (Citizen/Pro)
  - Form validation with input icons and loading states

- **Citizen Dashboard**
  - Personalized dashboard with gamification
  - Legal Passport section with badges
  - XP progression system with levels
  - Recent activities and favorite guides

- **Pro Dashboard**
  - Legal health indicator with percentage
  - Obligations timeline with deadlines
  - Quick tools (Simulator, Contract Generator)

- **Virtual Legal Advisor**
  - AI chat interface
  - Suggested questions
  - Voice input button (UI only)

- **Administrative Procedures Guide**
  - Step-by-step procedure navigation
  - Location maps with opening hours
  - Document checklist per step

- **Dashboard Layout**
  - Collapsible sidebar navigation
  - Mobile responsive menu
  - User profile section

- **UI Components**
  - Dialog, Progress, ScrollArea, Avatar components
  - Custom color palette (Haylalana brand)
  - Animations and micro-interactions

### Technical
- Next.js 15.5 with App Router
- React 19.1
- TypeScript 5
- TailwindCSS 4
- Radix UI components
- next-intl for internationalization (FR/MG)
- next-pwa for Progressive Web App support
